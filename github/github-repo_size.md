## Github Repository Size

## Description
Adds the repository size next to the repository name on GitHub repository pages and search results.
Displays a human-readable size (B, KB, MB, ...) fetched from the GitHub API. Supports a personal access token for higher rate limits or private repository access and a configurable color for the size label.

## Installation
1. Install a userscript manager (Tampermonkey, Greasemonkey, Violentmonkey) in your browser.
2. Add the userscript from the raw URL:
   - https://raw.githubusercontent.com/CarpeNoctemXD/UserScripts/refs/heads/main/github/github-repo_size.user.js
3. Alternatively, copy the userscript file into your userscripts manager manually.

## Usage
- Visit GitHub repository pages or use GitHub search; the repository size will be shown next to repo names.
- Open the userscript menu (Tampermonkey/Greasemonkey menu) and use:
  - "Set GitHub Token" — paste a GitHub personal access token (recommended if you hit rate limits or need private repo sizes).
  - "Set Repo Size Color" — provide a hex color (e.g., #ff9900) for the size label.
- The script auto-detects single-page navigation and updates sizes when the URL changes.

## Features
- Fetches and displays repository sizes from the GitHub API.
- Human-readable sizes (B, KB, MB, GB, ...).
- Menu options to set a GitHub token and customize the size label color.
- Works on repo pages and search results.
- Observes DOM changes and updates on client-side navigation.

## Configuration
- GitHub Token
  - Menu: "Set GitHub Token"
  - Purpose: Increases API rate limits and enables private repo size access.
  - Default: none (empty). Token must have the appropriate scopes if you need private repo info (e.g., repo).
- Repo Size Color
  - Menu: "Set Repo Size Color"
  - Format: Hex color string (e.g., #4493f8)
  - Default: #4493f8
